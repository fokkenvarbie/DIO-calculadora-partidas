<h1>Calculadora de Partidas Rankeadas</h1>

Um programa em JavaScript que calcula o saldo de partidas Rankeadas de um jogador e classifica seu nível com base na quantidade de vitórias. O sistema permite definir vitórias e derrotas para cada jogador e exibe de forma organizada o resultado no console.

- **Funcionalidades:**
  - **Cálculo de saldo:** Subtrai derrotas de vitórias para determinar o saldo de partidas.
  - **Classificação por níveis:** Define o nível do jogador (Ferro, Bronze, Prata, Ouro, Diamante, Lendário ou Imortal) com base no saldo de vitórias.
  - **Estruturas de decisão:** Utiliza condicionais para determinar o nível correto.
  - **Função reutilizável:** Permite calcular o nível de qualquer jogador passando vitórias e derrotas como parâmetros.
  - **Saída clara:** Exibe no console o saldo de vitórias e o nível correspondente de forma organizada.
